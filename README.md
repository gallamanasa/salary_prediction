Project Overview
This notebook implements a basic Linear Regression model to establish a relationship between an individual's years of experience and their corresponding salary. The goal is to predict salary for new individuals based on their experience. It covers data loading, preprocessing, model training, evaluation, and visualization of the results.

Dataset
The dataset used is Salary_dataset.csv, which contains two main columns:

YearsExperience: The number of years of professional experience.
Salary: The corresponding salary.
